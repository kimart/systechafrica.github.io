### Audit Trail

Clicking the Audit Trail link will open the Audit Trail window which lists all key activities that have taken place in the system per the active users.

A user can view details of audit trails, filter through the records and view print layout copies of records as shown in the screenshot below:

<img  alt="systems rights image" width="95%" height="auto"  class="center"  src="../media/adfd12.png">  
 
 **Action**

-  Click label 1 button to view the details of a selected user trail.
-  Click label 2 button after typing keywords to search for a record in the list.
-  Click label 3 button to repopulate the list after using the filter.
-  Click label 4 button to search records based on dates.
-  Click label 5 button to view a copy of print layout copy of audit trail.
